# Third Eye - AI Based Forensic Text-to-Image Generation System

## Description
Third Eye is an AI-powered forensic application that generates suspect face images from eyewitness descriptions.

## Features
- User Registration & Login
- AI Face Generation
- Suspect History
- Search & Delete Records
- SQLite Database
- Responsive Dashboard

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Render
- GitHub
